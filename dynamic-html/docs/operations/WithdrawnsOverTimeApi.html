withdrawnsPeerPeerHashIdTopGet

This Withdrawns Endpoint for a given peer, return the top prefixes by number of prefixes withdrawn in past 'hours'. Limit defaults to 25 and hours defaults to 2.\n

URL

https://api.openbmp.org/v1/withdrawns/peer/{peerHashId}/top

HTTP Method

GET

Response Type

List

Parameters

  • Path: peerHashId String

    The peerHashId value

  • Query: limit Integer

    The Limit Value in SQL query

  • Query: hours Integer

    The value in hours

  • Query: timestamp String

    The timestamp string
