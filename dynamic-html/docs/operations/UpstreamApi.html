This is the API

upstreamAsnCountGet

This upstream Endpoint returns all the upstream ASN's for the given ASN and count the number of distinct prefixes.\n

URL

https://api.openbmp.org/v1/upstream/{asn}/count

HTTP Method

GET

Response Type

List

Parameters

  • Path: asn Integer

    ASN Number
