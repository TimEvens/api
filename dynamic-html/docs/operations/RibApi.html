ribAsnAsnCountGet

This Rib Endpoint returns all prefixes by Origin ASN.\n

URL

https://api.openbmp.org/v1/rib/asn/{asn}/count

HTTP Method

GET

Response Type

List

Parameters

  • Path: asn Integer

    ASN Number
