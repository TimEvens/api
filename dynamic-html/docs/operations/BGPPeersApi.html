peerAsnAsnGet

This Bgp Peers Endpoint returns a list of BGP peers by Peer ASN.\n

URL

https://api.openbmp.org/v1/peer/asn/{asn}

HTTP Method

GET

Response Type

List

Parameters

  • Path: asn Integer

    ASN Number

  • Query: limit Integer

    The Limit Value in SQL query

  • Query: withGeo Boolean

    Boolean Flag

  • Query: where String

    The Where clause in SQL query

    The Order By clause in SQL query
