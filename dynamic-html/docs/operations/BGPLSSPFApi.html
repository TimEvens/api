linkstateSpfPeerPeerHashIdIsisRouterIdMtIdGet

This BGP-LS SPF Endpoint runs IS-IS SPF and generates a RIB for the given bgp-ls peer and router-id. The generated RIB is returned.\n

URL

https://api.openbmp.org/v1/linkstate/spf/peer/{peerHashId}/isis/{routerId}/{mtId}

HTTP Method

GET

Response Type

List

Parameters

  • Path: peerHashId String

    The peerHashId value

  • Path: routerId String

    The routerId value

  • Path: mtId Long

    The mtId value

  • Query: limit Integer

    The Limit Value in SQL query

  • Query: where String

    The Where clause in SQL query

  • Query: orderBy String

    The Order By clause in SQL query
