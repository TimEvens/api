This is the API

routersGet

This Routers Endpoint returns a list of BMP routers currently known in the database.\n

URL

https://api.openbmp.org/v1/routers

HTTP Method

GET

Response Type

List

Parameters

  • Query: limit Integer

    The Limit Value in SQL query

  • Query: withGeo Boolean

    Boolean Flag

  • Query: where String

    The Where clause in SQL query

  • Query: orderBy String

    The Order By clause in SQL query
