This is the API

downstreamAsnCountGet

This downstream Endpoint returns all the downstream ASN's for the given ASN and count the number of distinct prefixes.\n

URL

https://api.openbmp.org/v1/downstream/{asn}/count

HTTP Method

GET

Response Type

List

Parameters

  • Path: asn Integer

    ASN Number
