This is the API

asnStatsAsnGet

This As Statistics Endpoint returns the AS statistics for the provided ASN.\n

URL

https://api.openbmp.org/v1/asn_stats/{asn}

HTTP Method

GET

Response Type

List

Parameters

  • Path: asn Integer

    ASN Number

  • Query: limit Integer

    The Limit Value in SQL query

asnStatsGet

This As Statistics Endpoint returns a list of all active ASN's in the DB.\n

URL

https://api.openbmp.org/v1/asn_stats

HTTP Method

GET

Response Type

List

Parameters

  • Query: limit Integer

    The Limit Value in SQL query

asnStatsIpv4Get

This As Statistics Endpoint lists the top ASN's originating prefixes or that are transit to other ASN's.

URL

https://api.openbmp.org/v1/asn_stats/ipv4

HTTP Method

GET

Response Type

List

Parameters

  • Query: limit Integer

    The Limit Value in SQL query

  • Query: topTransit Integer

    The Limit Value for Transit and Order By Transit

  • Query: topOrigin Integer

    The Limit Value for Origin and Order by Origin
