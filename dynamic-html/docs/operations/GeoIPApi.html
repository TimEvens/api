This is the API

geoipIpAddressGet

This GEOIP Endpoint returns the geolocation data based on the IP supplied.\n

URL

https://api.openbmp.org/v1/geoip/{ipAddress}

HTTP Method

GET

Response Type

List

Parameters

  • Path: ipAddress String

    This could be localIP/ remoteIP/ routerIP/ peerIP

  • Query: where String

    The Where clause in SQL query
